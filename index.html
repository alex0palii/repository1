Hello GitHub

Команды создания и удаления теки:

  • cd или указать папку в текущей директории, или прописать полный путь: change directory (изменяет путь директории)
  • dir: directory (отображает список файлов выбранной теки)
  • mkdir имя теки || md имя теки: make directory (создает новую теку)
  • rmdir имя теки || rd имя теки: remove directory (удаляет теку, если она пустая)
  • rmdir /S имя теки || rd /S имя теки: remove directory (удаляет теку с файлами)

Команды создания и удаления файла:

  • copy con имя файла: создаёт новый файл, после введения текста нужно нажать CTRL+C
  • del имя файла: удаляет файл
  • type имя файла: выведет текст файла

Команды для GIT консоли:

  • git status: проверяет статус текущего коммита
  • git add: добавляет файл к коммиту для его отслеживания
  • git commit -m"Comment": создаёт коммит
  • git push: отправляет коммит в оттдалённый репозиторий
